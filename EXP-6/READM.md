# EXP-6.2 – Login Form with Validation
## Description
Implements a Login Form with custom validation logic using Regular Expressions. The form validates email format and enforces strong password rules with dynamic error messages.

## Components Used

- App.jsx

- LoginForm.jsx

- App.css

## Key Features

- Email format validation using Regex

- Password validation rules:

- Minimum 5 characters

- Must start with Capital letter

- Must contain at least one number

- Must contain at least one special character

- Dynamic error message display

- Form submission prevention using preventDefault()

- Success alert on valid login

- Node.js

- React
